# Требования к системе: Веб-сайт по продаже металла

> Версия: 1.0  
> Дата: 2026-03-13

Реализовано согласно спецификации в [need.md](need.md).

## Реализованный функционал

- **Архитектура**: Go backend + Vue 3 client + Vue 3 admin, Docker Compose (dev/prod)
- **Авторизация**: JWT, RBAC (admin/manager/editor)
- **Каталог**: CRUD, поиск, пагинация, категории, загрузка изображений, импорт/экспорт CSV/JSON
- **Контент**: Markdown-блоки для Главной и Контактов с санитизацией (bluemonday)
- **Заявки**: Форма обратной связи, honeypot, rate limiting
- **Продакшн**: PostgreSQL, Nginx, gzip, оптимизация изображений (resize + JPEG)
